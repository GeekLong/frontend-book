# word-clock

#### 介绍
用vue2.6构建一个抖音很火的【时间轮盘】小demo
设置功能： 国际化语言切换，全屏，垂直水平位置，缩放大小，旋转角度，文字颜色，背景颜色


#### 构建和设置

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

```
